Installation

npm install expressmod

Quick Start


  import expressmod from "expressmod";

  const app = expressmod();

  app.get("/", (req, res) => {
    res.send("Hello World");
  });

  app.listen(3000);

App

The app instance behaves like Express, but includes hardened defaults and reduced configuration overhead.

Routing

Routing syntax mirrors Express for compatibility.

Middleware

All middleware is automatically sandboxed and validated.

Built-in Protections

  • Request fingerprinting
  • Rate-limit guardrails
  • Header hardening
  • Input sanitization
